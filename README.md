### A Simple Magic 8 Ball Application

At the request of my daughter I built her a simple magic 8 ball app.

It calls on this fun and simple API and uses tkinter as a class to generate
the 8 ball.

This was a small, fun project me and my kid worked on so IDK if more updates will
come ... it is what it is. Reliable, Simple and Entertaining.
